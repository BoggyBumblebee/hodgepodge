Hodgepodge Help

Manage Homebrew with confidence.

Hodgepodge is a native macOS app for browsing Homebrew packages, checking local state, and running common maintenance tasks without losing the transparency of the underlying `brew` commands.

Quick Start

  1. Open the Overview screen to confirm Hodgepodge detected your Homebrew installation correctly.
  2. Use Refresh Homebrew to reload the executable path, version, prefix, cellar, repository, and tap count.
  3. Open other sections from the sidebar as they become available in later phases.

Troubleshooting

  • If Homebrew is not detected, install it first or make sure `brew` exists at `/opt/homebrew/bin/brew` or `/usr/local/bin/brew`.
  • If Help does not open, confirm the bundle contains `Help/index.html` and `help.css`.
  • If the app icon looks wrong, rebuild the app and confirm `Assets.car` is present in the final bundle.
